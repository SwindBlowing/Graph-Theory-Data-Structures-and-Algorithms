## Step 1 - 将有权边和有权图更改为模板类型

现实生活中的图可能具有各种类型的权重（比如说地图用实数表示距离，自动机用字符串表示转移），我们需要支持各种各样的权重。

例如：
```c++
template <typename T>
class WeightedEdge {
    // ......
};
```
所涉及的带权图，都需要使用模板类型。
